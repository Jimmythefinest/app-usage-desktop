# App Usage Vault

This vault stores desktop activity tracking data generated by [App Usage Clean](https://github.com/Jimmythefinest/app-usage-desktop).

## Vault Structure

```
Example_vault/
├── Activities/          Individual activity notes (auto-generated)
├── Sessions/            Work session notes (auto-generated)
├── Commits/             GitHub commit notes (auto-generated)
├── System/              Dashboards and analytics (Dataview-powered)
├── Scripts/             Custom JavaScript helpers
└── App Usage Obsidian intergration.md  Project overview and goals
```

## Note Types

| Type | Location | Purpose |
|------|----------|---------|
| Activity | `Activities/` | Single focused window or command usage event |
| Session | `Sessions/` | Grouped work sessions with time totals and commit links |
| Commit | `Commits/` | Linked GitHub commits from tracked sessions |
| System | `System/` | Live analytics dashboards and reports |

## System Dashboards

These notes are powered by Obsidian + Dataview. Open them to see live data from your vault.

- **Usage_By_Day** - Time spent per app for a selected date with share breakdown
- **Usage Trend** - Focused hours per day trend chart for a selected app
- **Recent Sessions** - Latest tracked sessions
- **GitHub Commits** - Recently synced commits

## Setup

1. Install the [Dataview plugin](https://github.com/micha/json-dictionary) in Obsidian
2. Run the importer with your vault path
3. Open the System dashboard notes to explore your data

## Editing

Activities and sessions are re-imported automatically. Do not edit activity notes manually unless `claimed: false`. You can add notes and outcomes to session notes freely.
